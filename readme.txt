实验器材:
	阿波罗STM32F429开发板
	
实验目的:
	综合测试实验,测试整个开发板硬件,体验STM32强大的功能.
	
硬件资源:
	1,DS0(连接在PB1),DS1(连接在PB0)
	2,串口1(波特率:115200,PA9/PA10连接在板载USB转串口芯片CH340上面)  
	3,串口3(波特率:115200,PB10/PB11)
	4,ALIENTEK 2.8/3.5/4.3/7寸LCD模块(包括MCU屏和RGB屏,都支持) 
	5,按键KEY0(PH3)/KEY1(PH2)/KEY2(PC13)/KEY_UP(PA0,也称之为WK_UP) 
	6,TPAD电容触摸按键(右下角LOGO,即TPAD,连接在PA5)
	7,RTC(实时时钟)
	8,DCMI接口(用于驱动OV5640摄像头模块)
	9,定时器TIM3,TIM4,TIM5,TIM6,TIM7,TIM8,TIM9
	10,PCF8574(控制OV_PWDN和蜂鸣器)
	11,LAN8720(通过RMII接口驱动)  
	12,SDRAM(W9825G6KH) 
	13,NAND FLASH(MT29F4G08) 
	14,ADC(读取内部温度传感器)
	15,WM8978(音乐播放/视频播放/NES游戏)
	16,光环境传感器(AP3216C)
	17,24C02(IIC连接在PH4/PH5上面) 
	18,W25Q256(接SPI5)
	19,触摸屏(TFTLCD模块自带)  
	20,DS18B20传感器一个(接在PB12上). 	
	21,MPU9250(和24C02共用IIC). 
	22,NRF24L01模块(SPI1(PB3/PB4/PB5)/IRQ(PG8)/CS(PG7)/CE(PG6)).  
	23,ALIENTEK OV5640摄像头模块
	24,USB_SLAVE接口(通过跳线帽将D-/D+连接在PA11/PA12上)
	25,USB_HOST接口(通过跳线帽将D-/D+连接在PA11/PA12上) 
	26,SD卡,通过SDIO(SDIO_D0~D4(PC8~PC11),SDIO_SCK(PC12),SDIO_CMD(PD2))连接 
	27,HS0038红外接收头
	28,板载咪头(录音)
	29,板载喇叭(播放音乐/NES游戏/视频播放)
	30,ALIENTEK SIM900/SIM900A GSM模块(电话/短信)
	
	
实验现象:
	本实验功能比较复杂,具体介绍请参考《STM32F429开发指南》,第六十七章。

	
注意事项:
	1,4.3寸和7寸屏需要比较大电流,USB供电可能不足,请用外部电源适配器(推荐外接12V 1A电源).
	2,本例程在LCD_Init函数里面(在lcd.c),用到了printf,如果不初始化串口1,将导致液晶无法显示!!  
	3,本例程需自备配件(非必须)包括:
		a,SD卡
		b,ALIENTEK OV5640摄像头模块 
		c,ALIENTEK GSM/GPRS模块
		d,DS18B20数字温度传感器
		e,NRF24L01无线模块
		f,耳机 
		g,USB游戏手柄/USB键盘一个 
	4,按RESET复位进入开机界面(自检界面)后,迅速按以下按键:
		按住KEY0不放:可以强制进入校准界面,对触摸屏进行校准(仅限电阻屏,对电容屏无效).
		按住KEY1不放:可以强制进入字库更新,更新字库.
		按住KEY2不放:可以强制擦除整个SPI FLASH,方便使用SD卡快速更新系统文件.
		---提示:按住KEY0~KEY2是要一直按住,并耐心等待.直到进入相应界面/出现提示.
	5,照相机功能,按KEY0/KEY2按键可以进行照相. 
	6,在音乐播放器界面,调节的音量不会自动保存,在设置界面设置好音量,并退回主界面,才会保存.	
	7,P9端口,跳线帽默认连接方式为:PB10(TX)连COM3_RX,PB11(RX)连COM3_TX.
	8,P11端口(多功能端口),默认用跳线帽连接:ADC和TPAD.
	9,P4端口,跳线帽连接方式为:PA9(T)连RXD,PA10(R)连TXD.
	10,P10端口,跳线帽连接方式为:PA11连D-,PA12连D+.
	11,无线传书功能,需要2套开发板才可以测试.
